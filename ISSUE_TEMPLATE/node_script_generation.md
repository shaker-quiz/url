## Задача: Создание скрипта Node.js для генерации файлов

**Описание:**
Создать и/или обновить все файлы `source/*/index.js` в корне репозитория на основе перечислений и вспомогательных функций, импортированных из существующего npm пакета `@shakerquiz/utilities`.

**Требования:**
1. Скрипт должен находиться в корне репозитория.
2. Все перечисления и функции-дополнения должны быть импортированы из пакета `@shakerquiz/utilities`.
3. Генерируемые файлы `index.js` должны соответствовать стандартам кода проекта.
4. Необходимо обрабатывать существующие файлы, обновляя их при необходимости.
5. Предусмотреть логику для добавления новых файлов, если это необходимо.

**Дополнительная информация:**
- Убедитесь, что у вас есть все необходимые зависимости для выполнения скрипта.
- Протестируйте скрипт на локальном окружении перед коммитом.
